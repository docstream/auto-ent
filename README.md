Another (m4) DSL to rule them all
=================================

_Work in progress_

Tool that generates code to stdout.
XML serializable features are annotated into output code.
..And optional ORM-stuff for the go output.

Depends on gnu{make,m4,sed}

Supports [for now] :

* Go (orm|xml)
* Java (JAXB)
* SQL (soon)
